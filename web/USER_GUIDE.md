# TalkRealm 前端使用指南

## 🎯 快速開始

### 1. 啟動應用程式

```bash
# 從專案根目錄執行
./scripts/start.sh
```

這會：
- 檢查並安裝依賴
- 執行資料庫遷移
- 建置並啟動伺服器
- 前端自動可用於 http://localhost:8080

### 2. 訪問應用程式

在瀏覽器中開啟：
```
http://localhost:8080
```

## 📖 功能教學

### 使用者註冊與登入

#### 註冊新帳號

1. 在首頁點擊「立即註冊」
2. 填寫以下資訊：
   - **電子郵件**：您的電子郵件地址
   - **使用者名稱**：3-32 個字元，用於系統識別
   - **暱稱**（選填）：顯示給其他使用者的名稱
   - **密碼**：至少 6 個字元
3. 點擊「註冊」按鈕
4. 註冊成功後會自動登入

#### 登入

1. 輸入註冊時使用的電子郵件和密碼
2. 點擊「登入」按鈕
3. 登入狀態會自動保存，下次訪問無需重新登入

### 社群管理

#### 建立社群

1. 點擊左側邊欄底部的 **+** 按鈕（綠色）
2. 輸入社群資訊：
   - **社群名稱**：您的社群名稱
   - **社群描述**（選填）：描述社群的用途
3. 點擊「建立」
4. 新建立的社群會出現在左側邊欄

#### 選擇社群

1. 在左側邊欄點擊任何社群圖示
2. 中間欄會顯示該社群的頻道列表
3. 右側會顯示社群成員列表

#### 社群功能

- **社群圖示**：顯示社群名稱的首字母
- **社群名稱**：顯示在頂部標題列
- **成員列表**：點擊右上角的使用者圖示可顯示/隱藏

### 頻道管理

#### 建立頻道

1. 選擇一個社群
2. 找到「文字頻道」或「語音頻道」分類
3. 點擊分類標題旁的 **+** 按鈕
4. 填寫頻道資訊：
   - **頻道類型**：文字或語音
   - **頻道名稱**：頻道的名稱
   - **頻道描述**（選填）：頻道的用途說明
5. 點擊「建立」

#### 頻道類型

- **文字頻道** 📝
  - 用於文字訊息交流
  - 圖示：# (hashtag)
  - 可以發送文字、圖片、檔案

- **語音頻道** 🔊
  - 用於語音通話（功能開發中）
  - 圖示：🔊 (speaker)
  - 目前僅支持文字訊息

#### 選擇頻道

1. 在頻道列表中點擊任何頻道
2. 頻道會被高亮顯示
3. 訊息區域會載入該頻道的歷史訊息

### 訊息功能

#### 發送訊息

1. 選擇一個頻道
2. 在底部輸入框中輸入訊息
3. 按 **Enter** 鍵發送
   - 或點擊右側的發送按鈕 ✈️

#### 訊息格式

每條訊息顯示：
- **使用者頭像**：左側圓形頭像
- **使用者名稱**：發送者的暱稱或使用者名稱
- **時間戳記**：
  - "剛剛" - 1 分鐘內
  - "X 分鐘前" - 1 小時內
  - "X 小時前" - 24 小時內
  - "X 天前" - 7 天內
  - 完整日期 - 超過 7 天
- **訊息內容**：實際的文字內容

#### 訊息分組

- 同一使用者在 5 分鐘內的連續訊息會被分組
- 分組的訊息不會重複顯示頭像和名稱
- 提供更清晰的閱讀體驗

#### 即時訊息

- 訊息通過 WebSocket 即時推送
- 無需重新整理頁面
- 訊息會自動顯示在頻道中
- 當前頻道的新訊息會自動捲動到底部

### 成員管理

#### 查看成員列表

1. 點擊頻道標題列右側的使用者圖示 👥
2. 右側邊欄會顯示所有社群成員
3. 再次點擊可隱藏成員列表

#### 成員資訊顯示

每個成員顯示：
- **頭像**：圓形頭像或預設圖示
- **狀態指示器**：
  - 🟢 綠色 - 線上
  - 🟡 黃色 - 離開
  - 🔴 紅色 - 忙碌
  - ⚫ 灰色 - 離線
- **暱稱**：成員的顯示名稱

### 使用者設定

#### 更新個人資料

1. 點擊左下角的設定圖示 ⚙️
2. 填寫要更新的資訊：
   - **暱稱**：更改顯示名稱
   - **頭像 URL**：設定頭像圖片連結
   - **狀態**：選擇您的狀態
     - 線上 🟢
     - 離開 🟡
     - 忙碌 🔴
     - 離線 ⚫
3. 點擊「儲存」

#### 登出

1. 點擊左下角的登出圖示 🚪
2. 確認登出
3. 會返回到登入頁面

### 左側邊欄 - 社群列表

- **🏠 首頁**：返回首頁視圖
- **社群圖示**：您加入的社群（顯示首字母或圖片）
- **分隔線**：區分不同區域
- **+ 建立社群**：建立新社群（綠色按鈕）

### 中間左側 - 頻道列表

- **社群標題**：當前選擇的社群名稱
- **⚙️ 社群設定**：管理社群（開發中）
- **頻道分類**：
  - 📝 文字頻道
  - 🔊 語音頻道
  - 點擊 **+** 建立新頻道
- **使用者面板**（底部）：
  - 頭像、名稱、狀態
  - ⚙️ 使用者設定
  - 🚪 登出

### 中間主要區域 - 訊息區

- **頻道標題列**：
  - 頻道名稱和描述
  - 🔔 通知設定
  - 📌 釘選訊息
  - 👥 顯示/隱藏成員列表
  - 🔍 搜尋訊息
- **訊息列表**：顯示所有訊息
- **輸入區域**（底部）：
  - ➕ 上傳檔案
  - 輸入框
  - 😀 表情符號
  - ✈️ 發送訊息

### 右側邊欄 - 成員列表

- **成員標題**："成員" 標題
- **成員列表**：顯示所有社群成員
- 每個成員顯示頭像、狀態和名稱

## 💡 使用技巧

### 1. 快速導航

- 使用滑鼠滾輪快速瀏覽訊息
- 點擊社群/頻道可快速切換
- 系統會記住您上次訪問的社群和頻道

### 2. 高效溝通

- 訊息會即時推送，無需手動重新整理
- 使用描述性的頻道名稱便於組織
- 建立不同用途的頻道分類

### 3. 團隊協作

- 為不同專案建立不同社群
- 在社群內建立主題頻道
- 邀請成員加入社群（功能開發中）

### 4. 個人化設定

- 設定有意義的暱稱
- 上傳個人頭像
- 使用狀態指示器告知可用性

## ⚠️ 常見問題

### 訊息沒有即時顯示？

1. 檢查 WebSocket 連接狀態（瀏覽器控制台）
2. 確認網路連接正常
3. 重新整理頁面
4. 重新登入

### 無法建立頻道？

1. 確認您是社群的擁有者或管理員
2. 確認已選擇正確的社群
3. 檢查頻道名稱是否有效

### 成員列表不顯示？

1. 點擊右上角的使用者圖示 👥
2. 確認您已選擇一個社群
3. 重新整理頁面

### 登入後白畫面？

1. 清除瀏覽器快取
2. 清除 localStorage（F12 > Application > Local Storage）
3. 重新登入
4. 檢查瀏覽器控制台錯誤訊息
